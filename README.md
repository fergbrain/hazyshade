hazyshade
=========
This is web app I designed for my Music Teacher circa 2007/2008. He wanted to be able to create a rehersal list based on who was singing what song. For example, let's say you have 5 people: Andrew, Bob, Chris, David, and Eddie. If Chris needs to practice Song A, but is also in Song B and F, then Song B and F cannot be sung, Mark needs to be able to quickly figure this out. Thus the following code.
